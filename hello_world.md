# 언어 별 Hello, World! 출력법

### C

```c
printf("Hello, World!");
```

### C++

```cpp
cout << ("Hello, World!") << endl;
```

### C`#`

```cs
System.Console.WriteLine("Hello, World!");
```

### Java

```java
System.out.println("Hello, World!");
```

### Javascript

```js
console.log("Hello, World!");
```

### PHP

```php
echo "Hello, world!";
```

### Go

```go
fmt.println("Hello, World!")
```

### Rust

```rust
println!("Hello, World!");
```

### Ruby

```rb
puts "Hello, World!"
```

### Python

```py
print("Hello, World!")
```

### R

```r
cat("Hello, World!")
```

### Julia

```julia
println("Hello, World")
```

### Matlab

```matlab
fprintf("Hello, World!");
```
